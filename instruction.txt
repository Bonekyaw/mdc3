** Expo Go 

1. Install nodejs LTS ( 22.12.0)
2. Install git 
3. Install vscode 
4. npx create-expo-app@latest ( via Terminal )
5. Open the project via vscode 
6. npx expo start 
*.  npm install ( If an error occurs )
*. npx expo start 
*. npx expo start --tunnel ( If QR scan error occurs )


***  Setting the set-executionPolicy to RemoteSigned in the Powershell in Windows OS

Set-ExecutionPolicy -Scope CurrentUser RemoteSigned -Force
